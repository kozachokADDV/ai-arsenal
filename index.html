// HOW IT WORKS

Как работает оркестрация

  1. 01

    Открытие сессии в рабочей папке

    Пользователь стартует Claude Code сессию, выбирая working directory под задачу (M1-Magazin / SEO_Developer / CFO_Plugin / другая).

  2. 02

    Загрузка контекста

    Claude автоматически читает: глобальный ~/.claude/CLAUDE.md + проектный ./CLAUDE.md + память проекта (журналы, профили, аудиты).

  3. 03

    Активация скилов или агентов

    По триггер-словам в запросе — активируются скилы. Для стратегических задач — вызывается диспетчер-агент через Agent tool.

  4. 04

    Делегирование специалистам

    Диспетчер (например seo-director) анализирует ситуацию, выбирает 1-3 подагентов, передаёт им задачу с полным контекстом.

  5. 05

    Результат + обучение

    Подагенты возвращают результаты диспетчеру. Тот синтезирует ответ. В конце сессии — записывает уроки в lessons-learned, обновляет patterns & site-profiles.
